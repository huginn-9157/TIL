## DS、チャート

- チャート一つに対してデータソースは一つ

    - チャートを２つ用意してレイヤーする
- [x] コンボチャートで任意の棒を列にするには
  1. 右クリック -> 列スタイルの変更
  2. チャート編集 -> スタイル設定 -> 線種表示設定
- [x] RFM分析で、購買日にあたる項目を取得できない
  - MB上で型を日時に指定しなければならない
- [x] チャートの行軸名が表示したりされなかったり…
  - `チャート編集 -> 軸 -> 行軸の設定 -> 行軸名を表示する`
